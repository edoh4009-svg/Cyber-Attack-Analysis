# Cybersecurity Data Analysis

## Assignment Overview

This project analyzes cybersecurity incidents using Python, Pandas, Matplotlib, and Seaborn.

## Dataset Features Analyzed

- Motive
- Location

## Visualizations

### 1. Attack Frequency by Motive
A count plot showing the most common motives behind cyber attacks.

### 2. Attack Frequency by Location
A count plot showing the locations with the highest number of recorded cyber incidents.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
